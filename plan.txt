The Prompt I'd Give a Coding Agent
You are a Senior Staff Software Engineer and Software Architect. Build a production-ready, enterprise-grade full-stack web application for "Noventra Solutions", a modern IT agency that provides website development, web applications, hosting, maintenance, cloud solutions, branding, and technical consulting.

## Core Requirements

Build the project using:
- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Framer Motion
- FastAPI
- PostgreSQL
- Prisma (frontend ORM if applicable) or SQLAlchemy/Alembic on the backend
- Redis
- JWT Authentication with Refresh Tokens
- Role-Based Access Control (RBAC)
- Docker for local development
- Clean Architecture with modular feature-based structure
- REST APIs with OpenAPI documentation
- Responsive, accessible (WCAG), SEO-friendly design

## Frontend

Create a premium SaaS-style UI inspired by Stripe, Linear, Vercel, Notion, and OpenAI. Use a purple-and-white brand theme for Noventra Solutions. The UI should be modern, fast, animated where appropriate, and fully responsive.

Public pages:
- Home
- About
- Services
- Service Details
- Portfolio
- Portfolio Details / Case Studies
- Live Demo Gallery
- Pricing
- Current Offers / Sales
- Testimonials
- Clients
- Blog
- Blog Details
- Careers
- Career Details
- FAQ
- Contact
- Privacy Policy
- Terms & Conditions
- Refund Policy
- Cookie Policy
- Sitemap

## Customer Portal

Authenticated customers can:
- View active projects
- Track project progress with milestone percentages and timeline
- View invoices and payment status
- Download deliverables
- Raise support tickets
- Use real-time live chat
- View contracts
- View hosting and domain renewal information
- Update profile and password
- Receive notifications

Implement centralized error handling with structured JSON responses. Include:

Use meaningful HTTP status codes and detailed error messages for both API and UI.
